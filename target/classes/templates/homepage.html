About the Project:

Overview

This application was created for CSCI 2235: Software Engineering with Dr. Kai Presler-Marshall. Over the 8 week project, we worked through the different stages in developing a full-stack Java application. The goal of the project was to create an online beverage ordering system to get caffeine-addicted computer scientists their fix!

Development Stages

In the first weeks, we began by familiarizing ourselves with the basic implementation of the coffee maker application and were tasked with resolving bugs and organizing our future modifications.

We then spent some time designing the backend of the program to meet the user requirements. During this stage, we worked on creating UML diagrams, dynamic flow diagrams, and API endpoint documentation. After solidifying our backend design, we proceeded with conceptualizing the UI experience using wireframes and static HTML pages.

With the majority of the design and organization complete, we began to implement the application. We used Spring Boot as the framework, SQL and Hibernate for our database management, Java for our middleware and controllers, and AngularJS and HTML for our frontend.

Development Practices

Throughout the project, we practiced test-first development using JUnit and continuous integration through GitHub. Thanks to our organized approach and consistent communication, our team was able to successfully create a functioning coffee maker application!
